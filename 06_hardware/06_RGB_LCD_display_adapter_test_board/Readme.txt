Instructions for Use
This PCB project must be used with the factory-provided source code supporting the ATK-DLIMX91S development board manufactured by ALIENTEK. 
The use of non-matching source code may cause abnormal functions or unstable operation. 
Any problems arising therefrom are not covered by the technical support of this PCB project.

Welding Instructions:
Component R3 is not required to be soldered. If R3 is soldered, the display screen may flash momentarily after the development board is powered on.
